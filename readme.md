# 🌸 Iris Flower Classification

A simple and interactive **Machine Learning web application** built with **Python** and **Streamlit** that predicts the species of an Iris flower based on its physical measurements.

## 🚀 Features

- 🌼 Predicts Iris flower species
- 📊 User-friendly Streamlit interface
- 🤖 Machine Learning powered classification
- ⚡ Instant predictions
- 📈 Displays prediction confidence (if implemented)
- 📝 Clean and responsive UI

[Click Here For Live Demo](https://iris-flower-classification-sus-tushhhh.streamlit.app/)

## 🌺 Iris Species

The model classifies flowers into one of the following species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

## 📊 Dataset

The project uses the famous **Iris Dataset**, which contains 150 samples with four features:

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

Target Classes:

| Class | Species         |
| ----- | --------------- |
| 0     | Iris Setosa     |
| 1     | Iris Versicolor |
| 2     | Iris Virginica  |

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Iris-Flower-Classification.git
```

### 2. Navigate to the project

```bash
cd Iris-Flower-Classification
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

---

## 🎯 How to Use

1. Launch the Streamlit application.
2. Enter the flower measurements.
3. Click the **Predict** button.
4. View the predicted Iris species.
